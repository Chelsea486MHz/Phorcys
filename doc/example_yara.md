# Example with Yara rules applied on binary file

Generate a dummy file:

`$ echo "{\"title\": \"Hello world\"}" | gzip -c | base64 > example`

With your own rules:

```
# rules.yar
rule r: dummy_rule
{
    strings:
        $my_text_string = "hello world" nocase
    condition:
        $my_text_string
}
```

Run *Phorcys*:

`$ phorcys_decode.py -b example -y rules.yar`
